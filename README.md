# Flins's Portfolio - Vibe Edition

This is the source code for my personal portfolio website, built with a focus on modern aesthetics, smooth animations, and an interactive user experience. It's a showcase of my skills in front-end development and my passion for "vibe coding".

**Live Demo:** https://flins-portfolio-vibe.vercel.app/



---

## ✨ Features

This portfolio isn't just a static page; it's a fully interactive experience.

-   **Cinematic Entry Sequence:** A carefully choreographed animation sequence that introduces the hero section, locking scroll until the intro is complete.
-   **Interactive Particle Background:** A dynamic canvas background where particles react to mouse movement, creating a constellation-like effect.
-   **Scroll-Reveal Animations:** Every section gracefully animates into view as the user scrolls, powered by Framer Motion.
-   **Real-time GitHub Contributions:** The contribution graph fetches and displays my actual, live GitHub activity.
-   **Forced Landscape & Fullscreen Mode:** On mobile, the site enforces a landscape view for a consistent, PC-like experience. A dedicated button allows users to enter a true, immersive fullscreen mode.
-   **Custom Animated Preloader:** A sci-fi inspired loading screen with a custom logo and spinner that runs before the site is revealed.
-   **Custom Trailing Cursor:** A smooth, springy cursor that follows the mouse, adding a premium feel to the navigation.
-   **Modern UI/UX:** Glassmorphism, neon glows, and consistent design language are used throughout the site.

## 🛠️ Getting Started

To run this project locally on your machine, follow these steps:

### Prerequisites

-   Node.js (v18.x or later recommended)
-   npm or yarn

### Installation & Setup

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/devflins2/portfolio.git
    ```

2.  **Navigate to the project directory:**
    ```bash
    cd portfolio
    ```

3.  **Install dependencies:**
    ```bash
    npm install
    ```

4.  **Run the development server:**
    ```bash
    npm run dev
    ```

5.  Open http://localhost:3000 in your browser to see the result.
